Nodes represent electricity power stations in US and two nodes are connected if there is a power transmission line between them. 

"power-lines.labels": actual names for each station, sorted by vertices ids in the graph
"power-lines.xy": longitude and latitude for each station, sorted by vertices ids in the graph

data sources:
https://hifld-geoplatform.opendata.arcgis.com/datasets/electric-substations
https://hifld-geoplatform.opendata.arcgis.com/datasets/electric-power-transmission-lines
